Detailed Description

A TCP server that handles HTTP requests This class is a subclass of QTcpServer and handles incoming HTTP requests. It listens for incoming connections on a specified port and invokes a callback function for each request based on the path and method specified in the request.

Note
This class does not handle multiple requests concurrently. It processes one request at a time.
